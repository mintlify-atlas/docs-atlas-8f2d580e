## Overview

The Company Settings module allows you to configure your business's legal identity, contact information, and tax details. This is essential for proper invoicing, reporting, and compliance with local regulations.

<Note>
  Company information appears on all invoices, receipts, and official documents generated by the system.
</Note>

## Company Configuration

### Required Fields

When creating or editing a company, the following fields are mandatory:

<ParamField path="businessName" type="string" required>
  Legal business name (Razón Social) - automatically converted to uppercase
</ParamField>

<ParamField path="nit" type="string" required>
  Tax identification number (NIT) for your business
</ParamField>

<ParamField path="verificationDigit" type="number" required>
  Single-digit verification code (0-9) associated with your NIT
</ParamField>

### Optional Fields

<ParamField path="email" type="string">
  Business contact email address
</ParamField>

<ParamField path="address" type="string">
  Physical address of your business location
</ParamField>

<ParamField path="image" type="string">
  Company logo (base64 encoded image)
</ParamField>

## Managing Companies

### Creating a New Company

<Steps>
  <Step title="Open Company Module">
    Navigate to the Company section from the Administration menu.
  </Step>
  
  <Step title="Click 'Nueva Compañía'">
    The "New Company" button is visible only if you have `canCreate` permission.
  </Step>
  
  <Step title="Fill Required Information">
    Enter the business name, NIT, and verification digit. These fields are mandatory.
    
    ```typescript
    {
      businessName: 'MI EMPRESA S.A.S',
      nit: '900123456',
      verificationDigit: 7,
      email: 'contacto@empresa.com',
      address: 'Calle 10 #43-21, Medellín'
    }
    ```
  </Step>
  
  <Step title="Save Company">
    Click "Guardar Compañía" to create the company record.
  </Step>
</Steps>

### Editing Company Information

<Info>
  Only users with `canEdit` permission can modify company information.
</Info>

To edit a company:

1. Locate the company in the table
2. Click the edit icon (pencil) in the "Gestión" column
3. Modify the fields as needed
4. Click "Guardar Compañía" to save changes

### Searching Companies

Use the search bar to filter companies by:
- Business name (Razón Social)
- Tax ID (NIT)

```typescript
// Search is case-insensitive and filters in real-time
const search = searchTerm.toLowerCase();
filteredCompanies = companies.filter(
  c => c.businessName?.toLowerCase().includes(search) || 
       c.nit?.toLowerCase().includes(search)
);
```

## Validation Rules

### Business Name
- Required field
- Automatically converted to uppercase
- Must not be empty

### NIT (Tax ID)
- Required field
- Must be provided before saving
- Typically 9-10 digits

### Verification Digit
- Required field
- Must be a single digit (0-9)
- Cannot be null

```typescript
if (!activeCompany.nit || 
    activeCompany.verificationDigit === null || 
    !activeCompany.businessName) {
  // Validation fails
  return 'NIT, dígito de verificación y razón social son obligatorios.';
}
```

## API Integration

The Company module integrates with the backend through the `MasterService`:

### Get Companies
```typescript
masterService.getCompanies().subscribe({
  next: (res) => {
    if (res.code === 0) {
      companies = res.data || [];
    }
  }
});
```

### Create/Update Company
```typescript
const payload = {
  option: 'CREATE',
  companyId: activeCompany.companyId ?? 0,
  userId: sessionUserId,
  nit: activeCompany.nit,
  verificationDigit: activeCompany.verificationDigit,
  businessName: activeCompany.businessName,
  email: activeCompany.email,
  address: activeCompany.address,
  image: activeCompany.image
};

masterService.createCompany(payload);
```

## Best Practices

## Troubleshooting

<AccordionGroup>
  <Accordion title="Cannot save company without required fields">
    **Error:** "NIT, dígito de verificación y razón social son obligatorios."
    
    **Solution:** Ensure all three required fields (NIT, verification digit, and business name) are filled before saving.
  </Accordion>
  
  <Accordion title="Company list not loading">
    **Error:** "Error al cargar los clientes desde el servidor."
    
    **Solution:** Check your network connection and verify that the backend API is accessible. Contact your system administrator if the problem persists.
  </Accordion>
  
  <Accordion title="Cannot see edit buttons">
    **Issue:** Edit and delete buttons are not visible.
    
    **Solution:** Verify that your user role has the appropriate permissions (`canEdit` or `canDelete`) for the "Compañías" module.
  </Accordion>
</AccordionGroup>
