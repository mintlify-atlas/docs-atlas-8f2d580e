<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-3">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get your Karma POS system up and running in minutes</p>
  
  <Steps>
    <Step title="Clone and Install">
      Clone the repository and install dependencies to get started with your local development environment.
      
      ```bash
      git clone https://github.com/Neider0608/pos-frontEnd.git
      cd pos-frontEnd
      npm install
      ```
    </Step>
    
    <Step title="Configure Your Environment">
      Set up your environment configuration and API endpoints. The system requires connection to your backend API server.
      
      <Info>
        The default API endpoint is configured at `https://localhost:7197/api/`. Update this in your service files as needed.
      </Info>
    </Step>
    
    <Step title="Run Development Server">
      Start the Angular development server and access your POS system in the browser.
      
      ```bash
      ng serve
      ```
      
      Navigate to `http://localhost:4200/` to see your application running.
    </Step>
    
    <Step title="Set Up Initial Configuration">
      Complete the initial setup wizard to configure your company, warehouses, and user permissions.
      
      [Learn more about initial setup →](/installation)
    </Step>
  </Steps>
</div>
