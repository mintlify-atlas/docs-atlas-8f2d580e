## Overview

The Karma POS Customer Management system allows you to maintain detailed customer records for both individual and corporate clients. Customer information is integrated with the POS system for quick selection during sales transactions.

## Customer Data Structure

### Customer Model

```typescript
interface Customer {
    id: number;
    companiaId: number;
    userId: number;
    identificationTypeId: number;
    document: string;
    firstName: string;
    middleName?: string;
    lastName: string;
    secondLastName?: string;
    phone?: string;
    email: string;
    
    // Location data
    city?: string;
    department?: string;
    country?: string;
    address?: string;
    
    // Corporate data
    isCompany: boolean;
    nit?: string;
    businessName?: string;
    
    // Audit fields
    createdAt?: Date;
    updatedAt?: Date;
}
```

### Individual vs Corporate

The `isCompany` flag determines the customer type:

- **Corporate**: Uses `businessName`, `nit`, and additional business fields

## Creating & Managing Customers

### Create New Customer

```typescript
openAddDialog() {
    this.currentCustomer = this.getEmptyCustomer();
    this.isEditing = false;
    this.showDialog = true;
}

private getEmptyCustomer(): Customer {
    return {
        id: 0,
        companiaId: this.companiaId,
        userId: this.userId,
        identificationTypeId: 1,
        document: '',
        firstName: '',
        middleName: '',
        lastName: '',
        secondLastName: '',
        phone: '',
        email: '',
        city: '',
        department: '',
        country: '',
        address: '',
        isCompany: false,
        nit: '',
        businessName: '',
        createdAt: new Date(),
        updatedAt: new Date()
    };
}
```

### Save Customer

```typescript
saveCustomer() {
    // Validate required fields
    if (!this.currentCustomer.firstName || 
        !this.currentCustomer.document || 
        !this.currentCustomer.email) {
        this.messageService.add({
            severity: 'warn',
            summary: 'Campos requeridos',
            detail: 'Debe completar los campos obligatorios.'
        });
        return;
    }

    const operation = this.masterService.CreateClient(this.currentCustomer);

    operation.subscribe({
        next: (res) => {
            if (res.code === 0) {
                this.messageService.add({
                    severity: 'success',
                    summary: 'Éxito',
                    detail: res.message || 'Operación realizada correctamente.'
                });
                this.showDialog = false;
                this.loadCustomers();
            }
        }
    });
}
```

<Note>
  The same method handles both creating new customers and updating existing ones.
</Note>

### Edit Customer

```typescript
editCustomer(customer: Customer) {
    this.currentCustomer = { ...customer };
    this.isEditing = true;
    this.showDialog = true;
}
```

### Delete Customer

```typescript
deleteCustomer(customer: Customer) {
    if (!confirm(
        `¿Deseas eliminar el cliente "${customer.firstName} ${customer.lastName}"?`
    )) return;

    this.masterService.DeleteClient(customer).subscribe({
        next: (res) => {
            if (res.code === 0) {
                this.customers = this.customers.filter(
                    (c) => c.id !== customer.id
                );
                this.filteredCustomers = [...this.customers];
                this.messageService.add({
                    severity: 'success',
                    summary: 'Eliminado',
                    detail: res.message || 'Cliente eliminado correctamente.'
                });
            }
        }
    });
}
```

<Warning>
  Deleting a customer is permanent and cannot be undone. Ensure the customer has no active transactions before deletion.
</Warning>

## Loading Customers

### Fetch Customer List

```typescript
loadCustomers() {
    this.masterService.getClients(this.companiaId).subscribe({
        next: (res) => {
            if (res.code === 0) {
                this.customers = res.data || [];
                this.filteredCustomers = [...this.customers];
            } else {
                this.messageService.add({
                    severity: 'warn',
                    summary: 'Advertencia',
                    detail: res.message || 'No se pudieron cargar los clientes.'
                });
            }
        },
        error: () => {
            this.messageService.add({
                severity: 'error',
                summary: 'Error',
                detail: 'Error al cargar los clientes desde el servidor.'
            });
        }
    });
}
```

## Search & Filtering

### Customer Search

Search across multiple fields:

```typescript
filterCustomers() {
    const term = this.searchTerm.toLowerCase();
    this.filteredCustomers = this.customers.filter((c) =>
        `${c.firstName} ${c.lastName} ${c.businessName ?? ''} ${c.document}`
            .toLowerCase()
            .includes(term)
    );
}
```

**Searchable fields:**
- First name
- Last name
- Business name (for companies)
- Document number

### Reset Filters

```typescript
resetFilters() {
    this.searchTerm = '';
    this.filteredCustomers = [...this.customers];
}
```

When processing a sale, customer information is attached:

```typescript
saveInvoice(invoice: Invoice) {
    const payload = {
        companiaId: this.companiaId,
        userId: this.userId,
        clientId: invoice.customer?.id || null,
        clientName: invoice.customer?.businessName || 
            `${invoice.customer?.firstName ?? ''}

            'Consumidor Final',
        clientNit: invoice.customer?.nit || 
            invoice.customer?.document || '',
        // ... other invoice data
    };
    
    this.posService.createInvoice(payload).subscribe({
        next: (res) => {
            // Handle response
        }
    });
}
```

<Info>

</Info>

Permissions are loaded on component initialization:

```typescript
ngOnInit() {
    const session = this.authService.getSession() as AuthSession;
    
    if (!session) {
        this.resetPermissions();
        return;
    }

    const { userId, companiaId } = session;

    this.loginService.getPermissions(userId, companiaId).subscribe({
        next: (permissions) => {
            this.permissions = permissions.data ?? [];
            this.applyPermissions();
        },
        error: () => this.resetPermissions()
    });
}

private applyPermissions(): void {
    const moduleName = 'Clientes';
    const permission = this.permissions.find((p) => p.module === moduleName);
    
    if (!permission) {
        this.resetPermissions();
        return;
    }
    
    this.canView = permission.canView;
    this.canCreate = permission.canCreate;
    this.canEdit = permission.canEdit;
    this.canDelete = permission.canDelete;
    this.canExport = permission.canExport;
}
```
