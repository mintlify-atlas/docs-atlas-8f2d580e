## Overview

The Karma POS system provides a comprehensive point-of-sale interface for retail operations. Built with Angular and PrimeNG, it offers barcode scanning, product selection, cart management, customer selection, and flexible payment processing.

## Core Features

### Barcode Scanning

The POS system supports real-time barcode scanning for quick product lookup:

```typescript
searchByBarcode() {
    if (!this.barcodeSearch.trim()) return;

    const product = this.products.find(
        (p) => p.barcode === this.barcodeSearch.trim()
    );

    if (product) {
        this.addProduct(product);
    } else {
        this.showToast('warn', 'No encontrado', 
            `No existe producto con ese código.`);
    }

    this.barcodeSearch = '';
    this.focusBarcodeInput();
}
```

**Features:**
- Instant barcode lookup
- Automatic focus management
- Product not found alerts
- Support for standard barcode formats

### Product Search

Multiple search methods are available:

- **Barcode Search**: Scan or manually enter product barcodes
- **Product Code Search**: Search by internal product codes
- **Name Search**: Find products by name (F2 hotkey)

```typescript
searchByProductCode() {
    if (!this.productCodeSearch.trim()) return;

    const product = this.products.find((p) => 
        p.code.toString() === this.productCodeSearch.toLowerCase() || 
        p.name.toLowerCase().includes(this.productCodeSearch.toLowerCase())
    );

    if (product) {
        this.addProduct(product);
    }
}
```

### Stock Management

The POS integrates with real-time stock reservation:

```typescript
reserveStock(item: CartItem, cantidad: number, existe: boolean) {
    this.posService
        .validateAndReserveStock({
            action: 'RESERVAR',
            productId: item.id,
            quantity: cantidad,
            companiaId: this.companiaId,
            userId: this.userId,
            facturaTempId: this.activeInvoice!.tempId
        })
        .subscribe({
            next: (res) => {
                if (res.data.success != 1) {
                    // Rollback if insufficient stock
                    item.quantity = item.prevQuantity;
                    this.showToast('warn', 'Stock insuficiente', 
                        `Solo hay ${res.data.stockDisponible} unidades más disponibles`);
                } else {
                    // Confirm the quantity change
                    if (!existe) {
                        this.activeInvoice?.items.push(item);
                    }
                    item.prevQuantity = item.quantity;
                    this.updateTotals(this.activeInvoice!);
                }
            }
        });
}
```

<Note>
  Stock is automatically reserved when products are added to the cart and released when removed or quantities are reduced.
</Note>

### Multi-Invoice Support

Handle multiple customers simultaneously:

```typescript
createNewInvoice(index: number) {
    const newInvoice: Invoice = {
        invoiceNumber: `FV-${this.invoiceCounter.toString().padStart(3, '0')}`,
        index: index,
        date: new Date(),
        customer: null,
        items: [],
        subtotal: 0,
        grossSubtotal: 0,
        generalDiscount: 0,
        detailDiscount: 0,
        tempId: uuidv4(),
        totalVat: 0,
        total: 0,
        paymentMethods: [{ type: 'cash', amount: 0 }]
    };

    this.invoices.push(newInvoice);
    this.activeInvoice = newInvoice;
    this.invoiceCounter++;
}
```

**Use Cases:**
- Serve multiple customers without losing their carts
- Quick switching between active invoices
- Each invoice maintains its own state

### Pricing Calculations

Automatic calculation of:

- **Gross Subtotal**: Total before discounts
- **Detail Discounts**: Per-product discounts
- **General Discount**: Overall percentage discount
- **VAT/IVA**: Configurable tax rate
- **Final Total**: Complete calculation with all factors

```typescript
updateTotals(invoice: Invoice): void {
    let grossSubtotal = 0;
    let totalDiscountDetail = 0;

    // Calculate per-item values
    for (const item of invoice.items) {
        const discountPercent = item.discount || 0;
        const discountValue = (item.price * item.quantity * discountPercent) / 100;
        const itemSubtotal = item.price * item.quantity;
        const itemTotal = itemSubtotal - discountValue;

        item.discountValue = discountValue;
        item.subtotal = itemSubtotal;
        item.total = itemTotal;

        grossSubtotal += itemSubtotal;
        totalDiscountDetail += discountValue;
    }

    const ivaPercentage = this.ivaRate / 100;
    const generalDiscountAmount = 
        (grossSubtotal - totalDiscountDetail) * (invoice.generalDiscount / 100);
    const netSubtotal = grossSubtotal - totalDiscountDetail - generalDiscountAmount;
    const totalVat = netSubtotal * ivaPercentage;

    invoice.grossSubtotal = grossSubtotal;
    invoice.subtotal = netSubtotal;
    invoice.detailDiscount = totalDiscountDetail;
    invoice.totalVat = totalVat;
    invoice.total = netSubtotal + totalVat;
}
```

## Payment Processing

The POS supports multiple payment methods:

- **Cash**: Direct cash payment
- **Card**: Credit/debit card payments
- **Transfer**: Bank transfers
- **Financed**: Installment payments with configurable terms
- **Other**: Additional payment types

```typescript
handleCheckout() {
    if (!this.activeInvoice) return;
    const invoice = this.activeInvoice;

    // Validate items exist
    if (invoice.items.length === 0) {
        this.showToast('warn', 'Sin productos', 
            'Agrega productos antes de procesar.');
        return;
    }

    // Validate payment amount
    const totalPayments = invoice.paymentMethods.reduce(
        (s, p) => s + (p.amount || 0), 0
    );
    if (Math.abs(totalPayments - invoice.total) > 0.01) {
        this.showToast('error', 'Pago incompleto', 
            'Verifica los montos.');
        return;
    }

    this.saveInvoice(invoice);
}
```
