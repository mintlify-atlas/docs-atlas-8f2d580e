## Overview

The Karma POS Sales & Invoicing system provides comprehensive invoice management, including viewing invoice history, generating PDF receipts, filtering by various criteria, and canceling invoices when necessary.

## Invoice Status

Invoices can have three statuses:

```typescript
statusOptions = [
    { label: 'APROBADA', value: 'APROBADA' },
    { label: 'RECHAZADA', value: 'RECHAZADA' },
    { label: 'ANULADA', value: 'ANULADA' }
];
```

- **APROBADA**: Successfully processed invoice
- **RECHAZADA**: Rejected invoice (payment failure, etc.)
- **ANULADA**: Canceled/voided invoice

## Loading Invoices

### Fetch Invoice List

Load invoices for a date range:

```typescript
loadInvoices() {
    let companyId = 1;
    this.invoices = [];
    
    this.posService.getInvoices(
        this.startDate, 
        this.endDate, 
        companyId
    ).subscribe({
        next: (res) => {
            this.invoices = res.data || [];
            this.filteredInvoices = [...this.invoices];
        },
        error: () => this.messageService.add({
            severity: 'error',
            summary: 'Error',
            detail: 'No se pudieron cargar las facturas.'
        })
    });
}
```

<Note>
  Invoices are loaded based on the selected start and end dates, allowing for flexible reporting periods.
</Note>

## Filtering & Search

### Apply Filters

```typescript
applyLocalFilters() {
    this.filteredInvoices = this.invoices.filter((inv) => {
        const matchText = !this.filterClient ||
            inv.client_Name?.toLowerCase().includes(
                this.filterClient.toLowerCase()
            ) ||
            inv.invoice_Number?.toString().includes(this.filterClient);

        const matchStatus = !this.filterStatus || 
            inv.invoice_Status === this.filterStatus;

        return matchText && matchStatus;
    });
}
```

**Filter Options:**
- Client name or document
- Invoice number
- Invoice status (Approved/Rejected/Canceled)
- Date range

### Clear Filters

```typescript
clearFilters() {
    this.filterClient = '';
    this.filterDate = null;
    this.filterStatus = null;
    this.filteredInvoices = [...this.invoices];
}
```

## Invoice Statistics

### Calculate Sales Totals

```typescript
getStats() {
    const stats = {
        totalAprobada: 0,
        countAprobada: 0,
        totalAnulada: 0,
        countAnulada: 0,
        totalRechazada: 0,
        countRechazada: 0
    };

    this.filteredInvoices.forEach((inv) => {
        const total = inv.invoice_Total || 0;
        const status = inv.invoice_Status?.toUpperCase();

        if (status === 'APROBADA') {
            stats.totalAprobada += total;
            stats.countAprobada++;
        } else if (status === 'ANULADA') {
            stats.totalAnulada += total;
            stats.countAnulada++;
        } else if (status === 'RECHAZADA') {
            stats.totalRechazada += total;
            stats.countRechazada++;
        }
    });

    return stats;
}
```

<Info>
  Statistics update automatically based on the current filter criteria, providing real-time insights.
</Info>

## Canceling Invoices

### Cancel/Void Invoice

```typescript
cancelInvoice() {
    let companyId = 1;
    
    this.confirmationService.confirm({
        message: `¿Está seguro de anular la factura 
                  ${this.selectedInvoice?.invoice_Number}?`,
        header: 'Confirmación',
        icon: 'pi pi-exclamation-triangle',
        accept: () => {
            this.posService.cancelInvoice(
                this.selectedInvoice?.id, 
                companyId
            ).subscribe({
                next: (res) => {
                    this.messageService.add({
                        severity: 'success',
                        summary: 'Éxito',
                        detail: `Factura ${this.selectedInvoice?.invoice_Number} 
                                 anulada correctamente`
                    });
                    this.loadInvoices();
                },
                error: (err) => {
                    this.messageService.add({
                        severity: 'error',
                        summary: 'Error',
                        detail: `No se pudo anular la factura 
                                 ${this.selectedInvoice?.invoice_Number}`
                    });
                }
            });
        },
        reject: () => {
            this.messageService.add({
                severity: 'info',
                summary: 'Cancelado',
                detail: 'Acción de anular cancelada'
            });
        }
    });
}
```

<Warning>
  Canceling an invoice changes its status to "ANULADA" and may affect inventory stock levels. Ensure this action is necessary before proceeding.
</Warning>

## PDF Generation

### Generate Invoice PDF

Two PDF formats are supported:

1. **Standard Invoice** (A4 format)
2. **POS Receipt** (Thermal printer format)

```typescript
handleDownload() {
    let detailDiscount = this.selectedInvoice?.items?.reduce(
        (sum, item) => sum + (item.discount_Value || 0), 0
    ) || 0;
    
    this.invoicePdfService.generate({
        invoiceNumber: this.selectedInvoice?.invoice_Number || '',
        date: this.selectedInvoice?.created_At || new Date(),
        customer: this.selectedInvoice?.customer || null,
        items: this.selectedInvoice?.items || [],
        subtotal: this.selectedInvoice?.subtotal || 0,
        detailDiscount: this.selectedInvoice?.items?.reduce(
            (acc, item) => acc + (item.discount_Value ?? 0), 0
        ) ?? 0,
        generalDiscount: this.selectedInvoice?.descuento_General || 0,
        grossSubtotal: this.selectedInvoice?.subtotal_Bruto || 0,
        discount: detailDiscount + 
            (((this.selectedInvoice?.subtotal || 0) - detailDiscount) * 
             (this.selectedInvoice?.descuento_General || 0)) / 100,
        total: this.selectedInvoice?.invoice_Total || 0,
        deliveryInfo: this.selectedInvoice?.delivery,
        paymentMethods: this.selectedInvoice?.payments || [],
        session: this.selectedCompany
    });
}
```

### POS Thermal Receipt

```typescript
handlePrint(): void {
    this.invoicePdfService.generatePOS({
        // Same parameters as handleDownload
    });
}
```

<Tip>
  Use `handleDownload()` for standard PDF invoices and `handlePrint()` for thermal receipt printing.
</Tip>

## Discount Calculations

### Total Discounts

Calculate combined discounts:

```typescript
getDiscounts() {
    const general = this.selectedInvoice?.descuento_General ?? 0;
    const itemsDiscount = this.selectedInvoice?.items?.reduce(
        (acc, item) => acc + (item.discount_Value ?? 0), 0
    ) ?? 0;

    return general + itemsDiscount;
}
```

**Discount Types:**
- **Detail Discount**: Per-item discounts
- **General Discount**: Overall percentage discount applied to subtotal

## Currency Formatting

### Format Currency Values

```typescript
formatCurrency(value: number): string {
    return new Intl.NumberFormat('es-CO', {
        style: 'currency',
        currency: 'COP',
        minimumFractionDigits: 0
    }).format(value);
}
```

<Note>
  All monetary values are formatted according to Colombian Peso (COP) standards.
</Note>
