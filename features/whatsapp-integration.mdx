## Overview

Karma POS includes a powerful WhatsApp multi-agent chat system that enables your team to handle customer conversations in real-time. The system supports multiple agents, conversation assignment, media attachments, and real-time message synchronization using SignalR.

## Architecture

### Components

The WhatsApp integration consists of three main components:

**WhatsApp Multi-Agent Component** (`whatsapp-multiagent.component.ts:58`)
- Main UI component for managing conversations
- Handles message display, sending, and real-time updates
- Manages agent assignments and conversation locking

**WhatsApp Service** (`whatsapp.service.ts:10`)
- HTTP API client for WhatsApp operations
- Manages conversations, messages, and agent assignments

**SignalR Service** (`signalr.service.ts:7`)
- Real-time communication hub
- Delivers instant notifications when new messages arrive
- Manages phone number group subscriptions

### 4. Message Handling

#### Loading Messages

Messages for a conversation are loaded dynamically:

```typescript
loadMessages(conversationId: number) {
  this.messages = [];
  this.loadingMessages = true;
  
  this.whatsappService
    .getMessagesByConversationId(conversationId)
    .subscribe({
      next: (res) => {
        this.loadingMessages = false;
        if (res.code === 0) {
          this.messages = res.data || [];
        }
      }
    });
}
```

**API Endpoint:** `GET /api/WhatsApp/GetMessagesByConversationId/{conversationId}`

**Reference:** `whatsapp-multiagent.component.ts:548`

#### Sending Messages

Send text messages and media files to customers:

```typescript
sendMessage() {
  if (!this.messageText && this.selectedFiles.length === 0) return;
  
  const formData = new FormData();
  formData.append('conversationId', 
    this.activeConversation!.id.toString());
  formData.append('senderUserId', this.userId.toString());
  formData.append('companiaId', this.companiaId.toString());
  formData.append('to', this.activeConversation!.clientPhone);
  formData.append('phoneNumberId', 
    this.selectedPhoneNumber!.phoneNumberId);
  formData.append('accessToken', 
    this.selectedPhoneNumber!.accessToken);
  
  if (this.messageText?.trim()) {
    formData.append('messageText', this.messageText);
  }
  
  // Attach multiple files
  this.selectedFiles.forEach((file) => {
    formData.append('files', file);
  });
  
  this.whatsappService.sendMessage(formData).subscribe({
    next: (res) => {
      if (res.code === 0) {
        this.messageText = '';
        this.selectedFiles = [];
        this.loadMessages(this.activeConversation!.id);
      }
    }
  });
}
```

**API Endpoint:** `POST /api/WhatsApp/SendMessage`

**Reference:** `whatsapp-multiagent.component.ts:375`

### 5. Real-Time Updates with SignalR

The system uses SignalR for real-time message notifications:

#### SignalR Connection

```typescript
async connect() {
  this.hubConnection = new signalR.HubConnectionBuilder()
    .withUrl(`${this.apiUrl}/hubs/whatsapp`, {
      withCredentials: true,
      transport: signalR.HttpTransportType.LongPolling
    })
    .withAutomaticReconnect()
    .build();
  
  // Listen for new message events
  this.hubConnection.on('HasNewMessages', (data) => {
    this.messageArrived$.next(data);
  });
  
  await this.hubConnection.start();
}
```

**Reference:** `signalr.service.ts:16`

#### Joining Phone Groups

```typescript
async joinPhoneGroup(phoneNumberId: string) {
  await this.connect();
  await this.hubConnection.invoke('JoinPhone', phoneNumberId);
}
```

## Media Attachments

### File Upload

Handle file attachments with validation:

```typescript
onFilesSelected(event: Event) {
  const input = event.target as HTMLInputElement;
  if (!input.files) return;
  
  const files = Array.from(input.files);
  
  for (const file of files) {
    // Maximum 5 files per message
    if (this.selectedFiles.length >= this.MAX_FILES) {
      alert('Máximo 5 archivos por envío');
      break;
    }
    
    // Maximum 10MB per file
    const sizeMB = file.size / 1024 / 1024;
    if (sizeMB > this.MAX_FILE_SIZE_MB) {
      alert(`El archivo ${file.name} supera los ${this.MAX_FILE_SIZE_MB}MB`);
      continue;
    }
    
    this.selectedFiles.push(file);
    
    // Generate preview for images
    if (file.type.startsWith('image/')) {
      const url = URL.createObjectURL(file);
      this.previewImages.push({ file, url });
    }
  }
}
```

```typescript
updateStatus(conv: Conversation): void {

  // Release lock when closing
```

## 24-Hour Messaging Window

Check if the 24-hour messaging window is still open:

```typescript
is24hWindowOpen(): boolean {
  const lastClientMessage = [...this.messages]
    .reverse()
    .find((m) => m.from === 'client');
  
  if (!lastClientMessage) return false;
  
  const lastTime = new Date(lastClientMessage.timestamp).getTime();
  const now = Date.now();
  
  const hoursDiff = (now - lastTime) / (1000 * 60 * 60);
  return hoursDiff <= 24;
}
```

## Permissions

The WhatsApp module respects role-based permissions:

```typescript
private applyPermissions(): void {
  const moduleName = 'Chats';
  const permission = this.permissions.find(
    (p) => p.module === moduleName
  );
  
  if (!permission) {
    this.resetPermissions();
    return;
  }
  
  this.canView = permission.canView;
  this.canCreate = permission.canCreate;
  this.canEdit = permission.canEdit;
  this.canDelete = permission.canDelete;
  this.canExport = permission.canExport;
}
```
